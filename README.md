# MAC Install (Manual)
1. Install Python 3.9.9 or higher
2. Download the zip in #releases or git-clone
4. Open terminal in directory and type ```pip install -r requirements.txt```
6. Once this is done run ```python setup.py py2app```
7. and magic happens
